# AI-Based Online Chatbot Ticketing System - Project Gallery

This gallery contains the screenshots of the AI-Based Online Chatbot Ticketing System (RailAI). The images below demonstrate the major modules and functionalities of the application.

---

# 1. Dashboard

### Image File
```
screenshots/Dashboard Page.png
```

### Description
The Dashboard serves as the home page of the RailAI application. It provides an overview of the railway support system by displaying ticket statistics such as Total Tickets, Open Tickets, Resolved Tickets, Critical Tickets, and AI Resolved Issues. It also allows users to navigate quickly to other modules and monitor overall operational performance.

![Dashboard](screenshots/Dashboard%20Page.png)

# 2. AI Assistant

### Image File
```
screenshots/AI Assistant.png
```

### Description
The AI Assistant module enables users to communicate with the intelligent chatbot for railway-related support. Users can ask queries, report issues, create support tickets, and receive AI-generated responses. The interface also displays AI capabilities, session statistics, and suggested queries for better user interaction.

![AI Assistant](screenshots/AI%20Assistant.png)

---

# 3. Tickets

### Image File
```
screenshots/Tickets Page.png
```

### Description
The Tickets module displays all support tickets raised by users. It provides detailed ticket information such as Ticket ID, passenger details, issue description, priority level, status, assigned department, AI recommendations, and ticket resolution options. Administrators can efficiently manage, update, and resolve railway support requests from this page.

![Tickets](screenshots/Tickets%20Page.png)

# 4. Analytics

### Image File
```
screenshots/Analytics Page.png
```

### Description
The Analytics module presents graphical insights into the railway support system. It displays ticket trends, resolution rates, department-wise ticket distribution, category-wise statistics, and overall operational performance. These visual analytics help administrators monitor system efficiency and make informed decisions.
