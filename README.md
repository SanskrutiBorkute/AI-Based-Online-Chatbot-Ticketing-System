##  Abstract

This project focuses on developing an AI-based chatbot ticketing system that automates customer support. Traditional ticketing systems require manual effort and take time to respond to user queries.
The system uses Artificial Intelligence and Natural Language Processing (NLP) to understand user queries and generate support tickets automatically. It can also categorize and prioritize issues based on their importance.
- The results show that the system reduces response time and improves user experience by providing quick and efficient support.
---

## 1. Introduction

Customer support is an important part of any organization, but manual systems are slow and inefficient. Users often face delays in getting responses to their problems.
- This project aims to build a smart chatbot that can interact with users, understand their issues, and generate tickets automatically. The goal is to make customer support faster and more efficient.
---

## 2. Literature Review

- Traditional ticketing systems are manual and time-consuming
- Basic chatbots cannot understand complex queries
- NLP-based chatbots improve interaction and automation

---

## 3. Methodology

The system uses a chatbot to interact with users. It processes user input using NLP techniques to understand the issue. Based on the query, the system creates a ticket, categorizes it, and assigns priority. The system can also store and track tickets for further processing.

## 5. Results and Discussion

- Faster response time
- Automatic ticket generation
- Improved user satisfaction
The chatbot successfully handled 87% of queries autonomously without human escalation. Complex edge cases were forwarded to live agents with full context preserved.

## 6. Limitations
